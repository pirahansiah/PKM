## Product Suite

### Core Platform
- **Knowledge Graph Engine**: Unified semantic search across all data sources
- **Reasoning Layer**: Automated inference and query answering over graph structures
- **Edge Deployment**: Zero-latency inference on local devices (Axelera, Hailo, TensorRT)

### Data Ingestion
- Multi-format connectors: PDFs, databases, APIs, unstructured text
- Automatic schema extraction and ontology alignment
- Incremental sync with change detection

### Visualization
- Interactive graph explorer with Markmap WebAssembly
- Timeline views for historical data evolution
- Export to multiple formats (JSON, CSV, GraphML)
